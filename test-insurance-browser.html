🧪 Insurance Calculation Browser Test

This page helps you test the insurance calculations directly in your browser.

⚠️ Important Instructions

  1. Open this page in a NEW TAB
  2. Keep your shipment tracker page open
  3. Copy-paste the test scripts in the shipment tracker console
  4. Do NOT run tests on this page - it won't work without Supabase

🔧 Step 1: Check if Functions are Loaded

Copy this code and paste it in your shipment tracker browser console:

// Check if functions are available console.log("🔍 Function Check:"); console.log("setupInsuranceCalculationListeners:", typeof setupInsuranceCalculationListeners); console.log("calculateInsuranceValues:", typeof calculateInsuranceValues); console.log("updateFieldWithAnimation:", typeof updateFieldWithAnimation); console.log("supabase:", typeof supabase); console.log("shipmentId:", typeof shipmentId, shipmentId); // Check if modal elements exist console.log("\n🔍 Element Check:"); console.log("Insurance modal:", !!document.getElementById('insurance-modal')); console.log("Value field:", !!document.getElementById('value')); console.log("Rate field:", !!document.getElementById('rate')); console.log("Amount field:", !!document.getElementById('amount'));

🔧 Step 2: Force Setup Event Listeners

If functions exist, run this to force setup the event listeners:

// Force setup event listeners if (typeof setupInsuranceCalculationListeners === 'function') { console.log("🎯 Forcing setup of event listeners..."); setupInsuranceCalculationListeners(); console.log("✅ Event listeners setup complete"); } else { console.error("❌ setupInsuranceCalculationListeners function not found"); }

🔧 Step 3: Test Manual Calculation

Set test values and force calculation:

// Set test values in insurance modal const testValues = { value: 100000, rate: 0.5, marine_perc: 0.1, war_perc: 0.05, asc_1: 100, fif_perc: 2, sts_perc: 1, stamp: 50 }; // Apply values to form fields Object.keys(testValues).forEach(key => { const field = document.getElementById(key); if (field) { field.value = testValues[key]; console.log(`✅ Set ${key}: ${testValues[key]}`); } else { console.warn(`⚠️ Field not found: ${key}`); } }); // Force calculation if (typeof calculateInsuranceValues === 'function') { console.log("🚀 Forcing calculation..."); calculateInsuranceValues(); } else { console.error("❌ calculateInsuranceValues function not found"); }

🔧 Step 4: Test Backend Function Directly

Test the Supabase function directly:

// Test backend function const testData = { input: { value: 100000, rate: 0.5, marine_perc: 0.1, war_perc: 0.05, asc_1: 100, fif_perc: 2, sts_perc: 1, stamp: 50 }, shipment_id: shipmentId || "test-id", user_id: "test-user" }; console.log("🚀 Testing backend function..."); supabase.functions.invoke('calculate-insurance', { body: testData }) .then(result => { console.log("✅ Backend function result:", result); if (result.data && result.data.success) { console.log("✅ Calculations:", result.data.calculations); console.log("📊 Expected Amount:", result.data.calculations.amount, "(should be 500)"); console.log("📊 Expected Grand Total:", result.data.calculations.grand_total); } else { console.error("❌ Backend function failed:", result.error); } }) .catch(error => { console.error("❌ Backend function error:", error); });

🔧 Step 5: Check Browser Cache

❌ If Functions Still Not Found:

  1. Clear browser cache: Press Ctrl+Shift+R or Ctrl+F5
  2. Hard refresh: Hold Shift while clicking refresh button
  3. Check network tab: Verify shipment-tracker.js is loading the latest version
  4. Check timestamp: Look for the file modification date in network tab

✅ Expected Working Flow

When Everything Works:

  1. Open insurance modal → Click Edit
  2. You should see console: 🎯 Setting up insurance calculation listeners...
  3. Enter Value: 100000 → Should trigger: 🔄 calculateInsuranceValues called
  4. Enter Rate: 0.5 → Amount field should auto-fill with 500
  5. Console should show: ✅ Calculations received: {amount: 500, ...}
  6. All calculated fields should update automatically

🆘 Troubleshooting

Common Issues:

  • Functions not defined: Clear cache, hard refresh
  • Fields not found: Make sure you're in Edit mode of insurance modal
  • Backend errors: Check Supabase function logs in dashboard
  • No calculations: Check console for errors, verify input values
